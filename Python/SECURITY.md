# Security Policy

## Supported Versions

Only the latest version of Specter is supported with security updates.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead, report responsibly via email:

security@specter.local

Include:
- Description of the issue
- Steps to reproduce
- Potential impact

## Scope

This policy applies to vulnerabilities in Specter itself, not misuse of the tool.

## Responsible Disclosure

Please allow time for maintainers to respond before public disclosure.
