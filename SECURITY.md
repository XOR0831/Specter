# Security Policy

## Supported Versions

This project is under active development. Security fixes are applied only to the
latest version available on the `main` branch.

| Version | Supported |
|--------|-----------|
| main   | ✅ Yes     |
| older releases | ❌ No |

---

## Reporting a Vulnerability

Please **do not open public issues** for security vulnerabilities.

Instead, report issues using:
- GitHub Security Advisories (preferred)

Include:
- Clear description
- Reproduction steps
- Potential impact
- Mitigation ideas (if any)

You will receive an acknowledgement within **72 hours**.

---

## Scope

Security considerations include:
- Network safety and scanning behavior
- Resource exhaustion (CPU, memory, sockets)
- Unsafe input handling
- Dependency vulnerabilities

---

## Disclaimer

Specter is intended for **educational and authorized testing only**.
